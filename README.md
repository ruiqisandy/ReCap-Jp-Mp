# ReCap-Jp-Mp
**A Chrome extension for the Google Chrome Built-in AI Challenge 2025.**  

A Chrome extension that imports a user's AI chat history (e.g., from ChatGPT), uses on-device AI to thematically label conversations, and synthesizes the knowledge into interactive tools like a hyperlinked mind map.
